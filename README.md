# 🎮 Krypton Sensi - Painel de Otimização

App Android nativo em **Kotlin** para otimizar o desempenho do seu dispositivo e melhorar o FPS em games como Free Fire.

## ✨ Funcionalidades

- ⚡ **Otimização Automática** - Limpa RAM, desativa apps em background
- 📊 **Monitoramento de Memória** - Exibe RAM em tempo real
- 🎨 **Interface Azul e Preta** - Design moderno e atraente
- 🎮 **Integração Free Fire** - Abre o jogo com 1 clique
- ⚙️ **Redução de Animações** - Melhora performance do sistema
- 🔧 **Permissões Automáticas** - Solicita tudo que precisa ao iniciar

## 📋 Requisitos

- Android 5.0+ (API 21)
- 50MB de espaço disponível
- Kotlin 1.9.0+

## 🚀 Compilação

1. Clone o repositório
2. Abra no Android Studio
3. Clique em "Sync Now"
4. Execute `Build > Make Project`
5. Rode em um dispositivo/emulador

## 📦 Gerar APK

```bash
# Release APK (otimizado)
Build > Build Bundle(s) / APK(s) > Build APK(s)

# Ou via Gradle
./gradlew assembleRelease
```

O APK estará em: `app/release/app-release.apk`

## 🎯 Como Usar

1. Instale o APK no seu dispositivo
2. Abra o app "Krypton Sensi"
3. Conceda as permissões solicitadas
4. Clique em "⚡ OTIMIZAR AGORA"
5. Aproveite o desempenho melhorado!
